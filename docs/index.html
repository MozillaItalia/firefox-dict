LE PAROLE CHE MANCANO AL TUO FIREFOX

Dict è un set personalizzato e collaborativo di termini corretti, o per lo meno adottati nella lingua italiana, che tenta di colmare la lacuna del controllo ortografico di Mozilla Firefox, basato sul porting del dizionario di OpenOffice (oggi LibreOffice), talvolta non al passo con i tempi.

persdict.dat

Cos'è

Firefox salva i termini da te segnalati come corretti all'interno del file persdict.dat (che si trova nella cartella del profilo in uso). Più profili di Firefox utilizzi (forse sparsi su più postazioni), più sarà facile dimenticarsi di clonare il dizionario personalizzato in ognuno di questi. Voglio provare a risolvere questo problema.

Sempre online

Aggiornamenti che vanno sempre a buon fine, a prescindere dal fatto che il sito ufficiale sia online o meno. FIREFOX:DICT utilizza lo spazio messo a disposizione da GitHub per tenere la copia più aggiornata del dizionario h24, 7 giorni su 7.

Una grande famiglia

Nasce per Firefox ma può essere utilizzato anche per altri prodotti della stessa famiglia, come SeaMonkey o Thunderbird, oppure per tutti gli altri browser basati su Firefox (per esempio Pale Moon o Waterfox, ma anche altri).

Compatibilità garantita

Il file persdict.dat è il medesimo su Windows, Linux e macOS. Puoi scaricarlo da GitHub e sostituirlo a quello che hai attualmente in qualsiasi momento, anche con Firefox già aperto.

Ancora più semplice su Windows

Se il tuo Sistema Operativo è basato su Windows, ho scritto un piccolo codice VBS che può occuparsi di tutto. Scaricherà da GitHub il file di dizionario più aggiornato, quindi lo sostituirà a quello già presente nel tuo profilo.

Uso più profili di Firefox distribuiti su diverse mie postazioni di lavoro. Ho pensato di unire le varie fonti creando e aggiornando un solo file dat (un po' come la scoperta dell'acqua calda, non credi?).

COME FUNZIONA / DOWNLOAD

Scarica, sostituisci, utilizza.

  • Spostati nella barra URL di Firefox, quindi digita about:profiles e premi invio.
  • Nella schermata che comparirà, individua il profilo in uso (lo noterai facilmente, sotto al nome del profilo troverai la dicitura "Non è possibile eliminare il profilo attualmente in uso."), quindi fai clic sul pulsante Apri cartella in corrispondenza di "Cartella radice" (su macOS troverai "Mostra nel Finder", NdA).
  • Scarica il file persdict.dat più aggiornato da GitHub (lo trovi nella colonna di sinistra, qui di fianco), quindi sostituiscilo a quello che hai nel tuo profilo di Firefox.

    • Il nuovo dizionario personalizzato inizierà a essere operativo al successivo avvio di Firefox (puoi continuare a usare Firefox senza riavviarlo, ma ti verranno segnalati errori ortografici probabilmente già "corretti" nel file di dizionario personalizzato).

    VBSCRIPT

    Su Windows puoi automatizzare il download.

    Utilizzando un semplice VBScript potrai scaricare il file di dizionario più aggiornato da GitHub e caricarlo all'interno della cartella profilo che stai utilizzando.
    Il VBScript è disponibile anch'esso su GitHub (fai clic qui). Scaricalo e aprilo con un editor di testo e modifica la riga che comincia con HTTPDownload:

    HTTPDownload "https://raw.githubusercontent.com/mozillaitalia/firefox-dict/master/persdict.dat", "C:\Users\TUONOME\AppData\Roaming\Mozilla\Firefox\Profiles\TUACARTELLAPROFILO\persdict.dat"

    Sostituisci "C:\Users\TUONOME\AppData\Roaming\Mozilla\Firefox\Profiles\TUACARTELLAPROFILO\persdict.dat" con la vera posizione del tuo file persdict.dat, quindi salva e chiudi il file.

    Salvo errori, facendo doppio clic sul VBScript, dovrebbe partire il download e la sotituzione del file di dizionario personalizzato del tuo Firefox. L'operazione dovrà essere ripetuta (solo il doppio clic per il download) ogni volta che verrà rilasciato un nuovo aggiornamento del dizionario personalizzato su GitHub.

Ti va di contribuire?

Se hai già aggiunto dei termini personalizzati al tuo file di dizionario, condividili con me! Aggiungerò i tuoi vocaboli a quelli proposti nel persdict.dat su GitHub così che tutti possano trarne vantaggio. Ti propongo alcuni metodi per collaborare al progetto.

Via Dropbox

Ho creato una cartella di Dropbox all'interno della quale puoi caricare il tuo persdict.dat. Esegui l'upload del tuo file di dizionario personalizzato, verrà visualizzato e integrato nel master.

Via Form

Puoi copiare e incollare il contenuto del tuo persdict.dat all'interno di un modulo già pronto che puoi trovare sul mio blog all'indirizzo gioxx.org/firefoxdict-add.

Via GitHub (pull)

Sai usare le Pull requests di GitHub? Se sì, creane una, sarò ben lieto di valutarla e approvarla per l'integrazione nel master principale di FIREFOX:DICT.

Via GitHub (issue)

Se le Pull requests non fanno per te ma hai un account GitHub, apri una richiesta di assistenza / bug direttamente nelle Issue!
